# Medical Image Classification using CNN

## Overview
This project implements a simple Convolutional Neural Network (CNN) for image classification.
It demonstrates a basic pipeline for medical image analysis and deep learning experiments.

## Motivation
The goal is to explore deep learning models that can be extended to medical imaging tasks such as:
- Cancer detection
- Radiology image analysis
- Clinical decision support systems

## Model
A lightweight CNN architecture implemented in PyTorch.

## Dataset
MNIST dataset (used as a proxy for medical imaging experiments).

## How to Run
```bash
pip install -r requirements.txt
python train.py
